# MindList

MindList是一款把「思维导图」和「任务清单」结合在一起的工具。你可以一边梳理结构，一边管理执行进度。

## 立即使用

- 在线网页版（直接打开）：https://vuhthj685.github.io/MindList/
- 安装包下载（Windows）：https://github.com/vuhthj685/MindList/releases/latest
- 项目仓库：https://github.com/vuhthj685/MindList

## 3 步开始

1. 打开在线版或安装客户端
2. 新建一个导图，按你的思路拆分节点
3. 勾选已完成任务，并定期导出备份

## 核心体验

- 导图与任务联动：在节点中管理待办，完成状态更直观
- 本地优先：打开即用，不依赖复杂部署
- 可视化编辑：支持常见导图编辑流程和主题切换
- 同时提供网页版与客户端，按你的使用场景选择

## 数据与隐私

- 网页版默认将数据保存在用户自己的浏览器本地
- 不会自动把用户数据上传到你的 GitHub 仓库
- 建议定期导出文件备份，避免因清理浏览器缓存导致数据丢失

## 下载与发布说明

- 在线版由 GitHub Pages 托管
- 客户端安装包通过 GitHub Releases 发布
